# MVP Scope

## In Scope

Phase 1 includes:

- Owner DID
- Institution DID
- Agent DID
- Ownership VC
- KYC/KYB VC
- Authority VC
- Policy VC
- account model
- mandate model
- payment intent
- resource intent
- stablecoin settlement
- challenge / trusted surface hooks

## Out of Scope

Phase 1 does NOT include:

- full trading venue
- exchange / orderbook
- advanced reputation market
- multi-chain architecture
- large consumer wallet app
- complex asset management workflows
- full AP2 or x402 integration
- production-grade fiat bridge

## Primary MVP Scenarios

1. Agent-to-agent API/tool payment and settlement
2. Merchant agent paying service agents
