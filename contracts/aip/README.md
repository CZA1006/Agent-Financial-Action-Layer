# Contracts AIP Interface Notes

## Phase 1 Goal

Define interface-level notes for identity and credential status anchoring only.

## Candidate Surfaces

- DID anchor or registry interface
- credential status / revocation anchor interface
- identity freeze / revoke status surface

## Notes

- keep full DID resolution off-chain first
- only anchor narrow state that benefits from public verification
- do not build a complete on-chain DID method in Phase 1
